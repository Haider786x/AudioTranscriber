# Audio Transcriber

A full-stack audio transcription web app built with **Spring Boot** and **React**, using **AssemblyAI** for cloud-based speech-to-text.

## Live
- Frontend: https://audiotranscribe.netlify.app  
- Backend: https://audiotranscriber-9jau.onrender.com

## Tech Stack
- Frontend: React (Vite), Axios
- Backend: Spring Boot, REST APIs
- Cloud STT: AssemblyAI
- Deployment: Netlify (frontend), Render (backend), Docker

## Features
- Upload audio files and get transcriptions
- Responsive UI (mobile + desktop)
- Secure CORS-enabled backend
- Monorepo setup (Frontend + Backend)
